# Selected Test Scenarios (Education Domain)

Answering a factual question

Summarizing a concept

Providing advice/recommendations

# Scenario 1: Generating a Creative Story
# Naïve Prompt (Basic):

"Write a story about technology in education."

# Zero-Shot Prompt:

"Without any examples, write a short story about how technology helps students in a classroom."

# Few-Shot Prompt (2 Examples):

Example 1: Tablets help students access digital textbooks.
Example 2: Online quizzes provide instant feedback to learners.
Now, write a new story about technology improving student learning in school.

# Chain-of-Thought Prompt:

Step 1: A school introduces new educational technology.
Step 2: Students begin using digital tools for learning.
Step 3: Teachers monitor progress using data.
Step 4: Students improve performance over time.
Step 5: Write a story combining these steps.

# ChatGPT Response:

"In a small town school, teachers introduced smart tablets in every classroom. At first, students were curious and excited. Soon, they began using interactive lessons and online quizzes that gave instant feedback. Teachers tracked their progress through learning dashboards and offered extra help where needed. Within months, grades improved, and students became more confident learners. Technology had transformed the classroom into an engaging and supportive space."

# Scenario 2: Answering a Factual Question
# Naïve Prompt:

"How does online learning work?"

# Zero-Shot Prompt:

"Without examples, explain how online learning platforms deliver education to students."

# Few-Shot Prompt (2 Examples):

Example 1: Video lectures allow students to learn remotely.
Example 2: Discussion forums enable peer interaction.
Now explain another way online learning supports education.

# Chain-of-Thought Prompt:

Step 1: Students need access to educational content.
Step 2: Platforms provide video lectures and materials.
Step 3: Students complete assignments and quizzes.
Step 4: Teachers provide feedback digitally.
Step 5: Explain the full process clearly.

# ChatGPT Response:

"Online learning works by delivering educational content through digital platforms. Students access video lectures, reading materials, and assignments using the internet. They complete quizzes and participate in discussions to reinforce learning. Teachers monitor progress and provide feedback online, making education accessible from anywhere."

# Scenario 3: Summarizing a Concept
# Naïve Prompt:

"Summarize blended learning."

# Zero-Shot Prompt:

"Without examples, summarize the concept of blended learning in education."

# Few-Shot Prompt (2 Examples):

Example 1: Traditional classroom teaching provides direct interaction.
Example 2: Online learning offers flexibility and digital resources.
Now summarize how blended learning combines these approaches.

# Chain-of-Thought Prompt:

Step 1: Identify key elements of classroom learning.
Step 2: Identify key elements of online learning.
Step 3: Explain how both are integrated.
Step 4: Provide a concise summary.

# ChatGPT Response:

"Blended learning is an educational approach that combines traditional classroom teaching with online learning methods. It integrates face-to-face interaction with digital resources and flexible learning options, enhancing student engagement and accessibility."

# Scenario 4: Providing Advice / Recommendations
# Naïve Prompt:

"Give advice on improving student performance."

# Zero-Shot Prompt:

"Without examples, suggest ways teachers can improve student academic performance."

# Few-Shot Prompt (2 Examples):

Example 1: Provide regular feedback on assignments.
Example 2: Encourage active participation in class.
Now suggest three more ways to improve student performance.

# Chain-of-Thought Prompt:

Step 1: Identify common challenges students face.
Step 2: Suggest teaching strategies to address these challenges.
Step 3: Explain how each strategy improves learning.
Step 4: Provide structured recommendations.

# ChatGPT Response:

"Teachers can improve student performance by offering personalized feedback, using interactive teaching methods, and setting clear learning goals. Incorporating technology tools and encouraging collaborative learning can further enhance engagement. Regular assessments help track progress and identify areas for improvement."

# Analysis
Naïve prompts generate acceptable but general responses.

Zero-shot prompts improve clarity but may lack depth.

Few-shot prompts improve contextual understanding and relevance.

Chain-of-Thought prompts produce structured, logical, and more detailed outputs.

Refined prompts consistently improve clarity, completeness, and reasoning depth.

# Example 1: Educational Explanation
Broad Prompt

Prompt:

Explain photosynthesis.

Typical Model Response:

General definition

No target audience

Limited structure

Key steps may be missing

Refined Prompt

Prompt:

Explain photosynthesis in simple terms for a Class 6 student. Include steps and one example.

Typical Model Response:

Simple language

Step-by-step explanation

Clear example

Well-structured and accurate

✔ Better quality, accuracy, and depth

# Example 2: Problem Solving (Math)
Broad Prompt

Prompt:

Solve this equation: 2x + 5 = 15

Typical Model Response:

Gives answer directly

No explanation

Refined Prompt

Prompt:

Solve the equation 2x + 5 = 15 step by step and explain each step.

Typical Model Response:

Step-by-step solution

Clear reasoning

Final answer highlighted

✔ Improved depth and clarity

# Example 3: Programming (Java)
Broad Prompt

Prompt:

Write a Java program to check even or odd.

Typical Model Response:

Basic code

No input handling

No explanation

Refined Prompt

Prompt:

Write a Java program to check whether a number is even or odd using Scanner input. Include comments.

Typical Model Response:

Complete program

Uses Scanner

Comments included

Proper output

✔ Higher technical accuracy

# Example 4: Creative Writing
Broad Prompt

Prompt:

Write a story about friendship.

Typical Model Response:

Creative but unfocused

No clear theme or length

Refined Prompt

Prompt:

Write a 150-word story about friendship between two school students, with a moral at the end.

Typical Model Response:

Controlled creativity

Clear theme

Moral included

✔ Better focus and consistency

# Example 5: Opinion / Analysis
Broad Prompt

Prompt:

Discuss online education.

Typical Model Response:

One-sided or shallow

No structure

Refined Prompt

Prompt:

Discuss online education by explaining two advantages and two disadvantages in short paragraphs.

Typical Model Response:

Balanced answer

Structured points

Relevant examples

✔ Better analytical quality
